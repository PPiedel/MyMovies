# MyMovies
Kotlin mobile app written with the us of MVVM pattern. TMDb API REST client. 


#Features : 
- movies list (top rated, popular, upcoming, now playing)
- movie details

# Technologies : 
- Kotlin
- RxJava
- RxAndroid
- Dagger 2
- Retrofit 2
- TMDb API
- JUnit
- Mockito
- Robolectric
- Espresso
