# MyMovies
Kotlin Android app written with the usage of MVVM pattern. The Movie Database API REST client. 


# Features : 
- movies list (top rated, popular, upcoming, now playing)
- movie searching
- movie details


# Technologies : 
- Kotlin
- RxJava
- Dagger 2
- Retrofit 2
- JUnit
- Mockito
- Robolectric

# Design patterns : 
- MVVM Pattern
- Repository Pattern
